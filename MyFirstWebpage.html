WEB DESIGN: THE BASICS OF HTML

What is HTML?

Firstly, it must be clarified that HTML is not a programming language, but rather a markup language.
HTML stands for HyperText Markup Language and is the standard markup up language for developing web pages.

When considering creating a webpage, the construction of a house is an excellent example to examine,
wherein three key points arise:

1- The skeleton/framework/structure of the house
2- The design/look/aesthetics of the house that make it feel like a home
3- The functionality/operations that occur in the house that allow "things to work".

Likewise, when creating a webpage or website, HTML corresponds to the 1st key point,
CSS corresponds to the 2nd key point and JavaScript corresponds to the 3rd key point.

With regards to this webpage and it's creation, HTML will be placed in focus and expanded upon.
